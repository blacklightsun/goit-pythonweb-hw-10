# goit-pythonweb-hw-10
GoIT Neoversity module "FullStack Web Development with Python" Homework 10

Command for start (PostgreSQL + app):

```bash
docker-compose up -d
uv run uvicorn app.main:app --reload
```


Command for stop:
1. Ctrl-C on keyboard

2. Stop PostgreSQL container
```bash
docker-compose down
```
